# Quick Start
clone the repo  
```
git clone https://github.com/gaamarchi/mbr_parser.git
```
enter on dir
```
cd mbr_parser
```
run script  
  With shebang
  ```
  chmod +x mbr_parser.py
  ```
  
  ```
  mbr_parser.py -f mbr_file
  ```


  Without shebang
 ```
  python3 mbr_parser.py -f mbr_file
  ```
